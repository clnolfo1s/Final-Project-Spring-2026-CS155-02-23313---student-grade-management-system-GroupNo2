Project Description

This project is a Java-based Student Grade Management System that uses single-dimensional arrays to store and manage student data. The system allows users to add students, display all records, search for a student’s grade, calculate the class average, and determine the highest and lowest grades.

The program is menu-driven and demonstrates core Java programming concepts such as arrays, loops, conditionals, and methods.

Features:
Add student name and grade
Display all students and grades
Search for a student’s grade by name
Calculate class average
Find highest and lowest grades
Exit the program

Concepts Used:
Java 1D arrays
Loops (for, while)
Conditional statements (if, switch)
Methods
Input validation
Constraints
Maximum of 50 students
Grades must be between 0 and 100
No use of ArrayList or HashMap
Must handle invalid input gracefully

How to Compile and Run
Compile:
javac StudentGradeManagementSystem.java
Run:
java StudentGradeManagementSystem
